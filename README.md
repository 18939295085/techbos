## 书单来了版本管理系统

### 简介

书单来了版本管理系统，是书单来了项目集成app管理信息，app更新管理和app反馈信息管理系统

###技术点

* Vuejs
  * Vue-Router
  * Vuex
  * Axios
* 组件库
  * Element-UI
  * 自定义组件
* 构建工具
  * Webpack
  * Gulp
